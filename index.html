前言

Java NIO(New IO) 是一个可以替代标准 Java IO API 的 IO API（从 Java 1.4 开始)，Java NIO 提供了与标准 IO 不同的 IO 工作方式。Java7 则改进为 NIO.2，提供了全面的文件 IO 以及基于异步的 Channel 的 IO。
