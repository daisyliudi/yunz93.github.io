VPS搭建Java开发环境

发表于 2017-06-12 | 分类于 教程

原本一直使用的是电脑上双系统来使用进行Linux的学习，可是不知道为什么Ubuntu总是会崩溃，隔三差五的装系统太影响生产力了。而且Ubuntu 有一种魔力，你总是会忍不住地想去美化它，你会试图去寻找各种在 windows 上使用的工具的linux 版。通常折腾了一通，还是啥都没学会。

最近看了一篇统一开发环境的文章很棒。作者提到，可以使用 ssh 连接 linux 虚拟机的方法进行开发环境的统一，不会浪费很多时间在工具的选择上。

因此，我干脆买了搬瓦工上的VPS服务器，以作为开发平台使用。拒绝了界面的 linux，学习起来确实快了很多，直接 Xshell 连接上就可以操作，连输入法都不用安装，确实很好用。

VPS

我所使用的是搬瓦工的 vps，安装的系统是 CentOS 6。因为目前国内的大厂所使用的 linux 服务器大部分是 CentOS，这样选择，有利于尽早熟悉以后工作可能使用的开发环境。

至于是选择 Ubuntu 还是 CentOS，见仁见智。不过我不想把时间浪费在工具的选择上，just do it。

至于 SSH 工具，推荐使用 Xshell，可以保存会话，功能强大。
